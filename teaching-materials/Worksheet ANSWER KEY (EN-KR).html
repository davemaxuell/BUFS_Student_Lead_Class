② Predict & Vote — Answers / 예측 정답

1Which uses MORE tokens — English or Korean?
영어와 한국어 중 어느 쪽이 토큰을 더 많이 쓰나요?
Korean / 한국어. e.g. "Artificial intelligence is changing the world." = 8 tokens (EN) vs 17 tokens (KR) ≈ 2.1×. Why / 왜: the tokenizer was trained mostly on English, and Korean is agglutinative (교착어) — particles & endings attach to stems — so Korean words shatter into more, smaller subword pieces (often byte-level). · 토크나이저가 영어를 주로 학습했고, 한국어는 교착어라 조사·어미가 붙어 더 작은 조각으로 잘게 나뉩니다.
2How many pieces will “internationalization” become?
“internationalization”은 몇 조각이 될까요?
Character / 문자: 20  |  Word / 단어: 1  |  Subword / 서브워드: 2 → international + ization Teaching point / 핵심: subword keeps frequent chunks whole and reuses them, so it stays short without an “unknown” word. · 서브워드는 흔한 덩어리를 통째로 재사용해, 길지 않으면서도 ‘모르는 단어’가 없습니다. (BERT WordPiece may split a bit differently, e.g. international + ##ization.)
3Will “한” (one block) equal “한” (ㅎ+ㅏ+ㄴ) to the computer?
“한”(한 글자)과 “한”(ㅎ+ㅏ+ㄴ 분해형)을 컴퓨터는 같다고 볼까요?
Raw compare (===): NO / 아니오 (1 code point U+D55C vs 3 code points U+1112 U+1161 U+11AB).
After normalization (NFC): YES / 예 — NFC composes them into the same single syllable. Teaching point / 핵심: same look, different bytes → search/login/dedup can silently fail; normalization fixes it. · 같은 모양·다른 바이트 → 검색·로그인·중복제거가 조용히 실패; 정규화가 해결.
4Rank by token cost (1 = cheapest): English, Korean, Chinese, Amharic.
토큰 비용 순(1 = 가장 쌈): 영어, 한국어, 중국어, 암하라어.
1) English (8)  →  2) Chinese (12)  →  3) Korean (17)  →  4) Amharic (72) Same sentence, all measured with GPT's tokenizer. Amharic (low-resource, ~9× English) is the punchline. · 같은 문장을 GPT 토크나이저로 측정. 저자원 언어 암하라어가 영어의 약 9배 — 핵심 메시지.

③ Exit Ticket — model answer / 모범 답안

Explain the “token tax” in 2 sentences. / ‘토큰세’를 두 문장으로.
EN: AI models read text as tokens and were trained mostly on English, so the same sentence in a language like Korean or Amharic is cut into many more tokens. More tokens means you pay more, wait longer, and fill the model's memory faster — so speakers of under-resourced languages get a worse, costlier deal for the same task. KR: AI 모델은 글을 토큰으로 읽고 주로 영어로 학습돼서, 같은 문장이라도 한국어·암하라어 같은 언어는 훨씬 많은 토큰으로 쪼개집니다. 토큰이 많으면 비용이 늘고, 느려지고, 모델의 기억(문맥)이 빨리 차므로 — 저자원 언어 사용자는 같은 일을 하고도 더 비싸고 나쁜 대우를 받습니다.
