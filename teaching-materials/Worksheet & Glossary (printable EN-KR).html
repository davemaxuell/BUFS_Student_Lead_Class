① Glossary / 용어집

Term한국어Plain meaning / 쉬운 뜻
NLP자연어처리Computers working with human language · 컴퓨터가 사람 언어를 다루는 일
LLM거대언어모델A huge model that predicts text · 글을 예측하는 거대한 모델
Token토큰A chunk the model reads (not always a word) · 모델이 읽는 덩어리(꼭 단어는 아님)
Tokenization토큰화Cutting text into tokens · 글을 토큰으로 자르기
Vocabulary어휘집The fixed list of tokens a model knows · 모델이 아는 토큰 목록
OOV / UNK미등록 단어A word not in the vocabulary · 어휘집에 없는 단어
Normalization정규화Standardizing text (case, spacing, Unicode) · 글 표준화(대소문자·공백·유니코드)
Subword서브워드Pieces between a letter and a word · 글자와 단어 사이 크기의 조각
BPE / WordPiece / SentencePiece—Subword algorithms (GPT / BERT / multilingual) · 서브워드 알고리즘
Embedding임베딩Turning a token into numbers · 토큰을 숫자로 바꾸기
Morpheme형태소Smallest meaning unit (key for Korean) · 가장 작은 의미 단위(한국어 핵심)
Agglutinative교착어Attaches particles/endings to stems (Korean) · 어간에 조사·어미가 붙는 언어
Low-resource language저자원 언어A language with little training data · 학습 데이터가 적은 언어
Context window문맥 길이Max tokens a model reads at once · 한 번에 읽는 최대 토큰 수
Pretraining / Fine-tuning사전학습 / 미세조정Train broadly / adapt to a task · 넓게 학습 / 과제에 맞게 조정
